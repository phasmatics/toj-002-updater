ToJ-002 Firmware Updater

手順

  1. ToJ-002（XIAO SAMD21）を USB ケーブルで PC に接続します。
  2. 下の「接続してアップデート」を押し、ポップアップから TinyUSB Serial (COMx) または Seeeduino XIAO（ブートローダー中）の シリアルポートを選択してください。 通常動作中のポートを選ぶと、自動的にブートローダーモードへ移行します。
  3. 自動移行しない場合は、XIAO SAMD21背面にあるリセットパッドを 素早く２回ショート させてブートローダーモードに入れてから、もう一度「接続してアップデート」を押してください。 オレンジの LED がゆっくり点滅していればブートローダーモードになっています。
どの COM ポートを選べばいい？（Windows）
  1. キーボードの Win + X →「デバイスマネージャー」を開く
  2. 「ポート (COM と LPT)」を展開する
  3. ブートローダー中（LED 点滅）: Seeeduino XIAO (COMx) または USB Serial Device (COMx)
  4. 通常動作中: TinyUSB Serial (COMx) （Web Serial のポップアップに表示される名前。MIDI デバイスではない方）
  5. ブラウザのポート選択ダイアログで、上記と同じ名前のポートを選ぶ

ポート選択はボタンを押すたびに1回だけです。失敗した場合は準備を整えてから、再度ボタンを押してください。

重要: アップデート中は Cursor のシリアルモニタ（PlatformIO Serial Monitor）を 閉じてください。COM ポートは同時に1つのアプリしか使えません。
